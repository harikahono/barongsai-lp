# Chenglie Barongsai Surabaya

Landing page profesional untuk jasa pertunjukan barongsai di Surabaya & Jawa Timur.

🔗 **Live:** [chengliebarongsaisurabaya.com](https://chengliebarongsaisurabaya.com)

## Tech Stack

- **HTML/CSS/JS** — vanilla, tanpa framework
- **GSAP** — animasi scroll reveal, counter, parallax
- **GLightbox** — gallery lightbox
- **Google Fonts** — Playfair Display, Plus Jakarta Sans, Ma Shan Zheng

## Fitur

- 🐉 Floating glassmorphism navbar
- 📱 Responsive (mobile-first)
- 🎭 Kanji ornaments & dragon SVG decorations
- 🎞️ 3D carousel gallery
- ❓ FAQ accordion
- 📊 Animated stats counter
- 🗺️ Embedded Google Maps
- 📲 WhatsApp booking CTA

## Struktur

```
├── index.html          # Landing page utama
├── styles.css          # Semua styling
├── script.js           # Animasi & interaksi
├── images/             # Gallery, hero, dragon SVGs
└── .gitignore
```

## Deploy

### Netlify (Drag & Drop)

1. Buka [app.netlify.com](https://app.netlify.com)
2. Drag folder project ke Netlify
3. Done 🚀

### Netlify (Git)

1. Connect repo `harikahono/barongsai-lp`
2. Build command: _(kosongkan)_
3. Publish directory: `/` (root)
4. Deploy!

## Local Development

```bash
git clone https://github.com/harikahono/barongsai-lp.git
cd barongsai-lp
# Buka index.html di browser
```
